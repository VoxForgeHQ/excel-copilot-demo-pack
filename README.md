# Excel Copilot Mastery — Companion Downloads

This repository provides companion downloads for the book "Excel Copilot Mastery": the demo workbook, sample datasets, templates, prompt cards, and errata referenced in the book.

Quick overview
- /workbooks/ — Excel workbook(s) used in the book (Excel-Copilot-Demo-Pack.xlsx)
- /datasets/ — CSV files used in examples and exercises
- /templates/ — Template files and ZIPs (MVP-30)
- /prompt-cards/ — Prompt card PDFs and quick-reference materials
- /downloads/ — Web "downloads hub" page for easy access
- /errata/ — Changelog and known issues

Download options
1. Visit the GitHub Pages site (https://voxforgehq.github.io/) or the repository Downloads hub at `/downloads/`.
2. Click the item you want (links point to file paths such as `/workbooks/Excel-Copilot-Demo-Pack.xlsx` or `/datasets/Finance_Data.csv`).
3. Or clone or download the repository and open files locally (e.g., open the workbook in Excel).

Accessibility & usage notes
- Files are provided in standard formats (XLSX, CSV, PDF, ZIP). If you rely on assistive technologies, CSV files can be opened with screen readers or imported into other tools.
- The demo workbook includes a Welcome sheet with step-by-step examples and named ranges intended to improve compatibility with Excel Copilot.
