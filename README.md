# excel-copilot-demo-pack

Repository for the Excel Copilot demo pack.

Folders:
- Datasets/     — sample datasets and CSVs used in demos
- Workbooks/    — finished Excel workbooks and examples
- Templates/    — Excel templates for reuse
- Prompt-Cards/ — short prompts, examples, and usage notes for Excel Copilot
- Errata/       — corrections, known issues, and change notes
