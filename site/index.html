How to Use

1. Download the demo pack workbook (includes all datasets pre-loaded), or download individual CSV files if you prefer separate files.

2. Open the workbook in Excel and follow the Welcome sheet instructions to get started. The Welcome sheet includes step-by-step examples and links to the prompt cards.

3. Use Excel Copilot with the provided prompts to explore the data and try the examples. For best results, open the workbook before invoking Copilot so sample data and named ranges are available.
